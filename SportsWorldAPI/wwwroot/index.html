Sportsworld API

Rest

    Athletes: "http://localhost:5180/api/athlete"
    Finance "http://localhost:5180/api/finance"
    Venues "http://localhost:5180/api/venue"

Athletes

  • Id: int
  • Name: string
  • Gender: string
  • Price: int
  • Image: string
  • PurchaseSatus: bool

Endpoints

  • GET /api/athlete
  • GET by id /api/athlete/{id}
  • GET by name /api/athlete/get-by-name/{name}
  • POST /api/athlete
  • PUT /api/athlete/{id}
  • DELETE /api/athlete/{id}

Finance

  • Id: int
  • MoneyLeft: int
  • NumberOfPurchases: int
  • MoneySpent: int

Endpoints

  • GET /api/finance
  • POST /api/finance/add-loan/{amount}
  • POST /api/finance/purchase-athlete/{athleteId}

Venues

  • Id: int
  • Name: string
  • Capacity: int
  • Image: string

Endpoints

  • GET /api/venue
  • GET by id /api/venue/{id}
  • GET by name /api/venue/get-by-name/{name}
  • POST /api/venue
  • PUT /api/venue/{id}
  • DELETE /api/venue/{id}
